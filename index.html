How to use

  1. Get the json file: paimon.moe -> Settings -> Export & Import Data -> Download Data
    2. (Do not change the file name i.e File name should be "paimon-moe-local-data....json")
  2. Upload the obtained JSON file to this site

Why is there an option to exclude 6 and 12 months?

  • The most recent data which is in both file and genshin can conflict and create multiple entries while importing
  • Few versions ago, genshin only stored 6 months of wish data which is now increased to a whole year (i.e 12 months)
